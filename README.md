 --> Install react router dom (npm i -D react-router-dom)
 --> Header
 --> Routing
 --> Login Form
 --> Sign up Form
 --> Form Validation 
 --> useRef Hook
 --> Firebase setup - Firebase (https://firebase.google.com/)
